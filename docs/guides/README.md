## 🔄 Proceso para Escribir una Guía

1. **Elige un tema específico** que merezca una guía
2. **Estructura**: Problema → Solución → Ejemplo → Validación
3. **Sé práctico**: Incluye código real
4. **Sé conciso**: Una guía debe tomar 5-15 minutos leer
5. **Linkea desde este README**
